# Marketing Materials for Sayakahwin.com

This directory contains HTML/CSS templates for social media marketing posters.

## How to use

1. Open the HTML files in a web browser (Chrome, Firefox, Safari).
2. The designs are scaled down to fit on a standard laptop screen.
3. To export as an image:
   - Use a screenshot tool (like "GoFullPage" extension or system screenshot).
   - Or, use the browser's "Capture node screenshot" in Developer Tools:
     1. Right-click the `.container` element -> Inspect.
     2. Press `Ctrl+Shift+P` (Cmd+Shift+P on Mac).
     3. Type "Capture node screenshot" and select it.
     4. This will download a high-quality PNG of the poster.

## Files

- `instagram_feed.html`: 1080x1350 (4:5 ratio) - Optimized for Instagram Feed.
- `tiktok_story.html`: 1080x1920 (9:16 ratio) - Optimized for TikTok, Instagram Reels, and Stories.

## Design Notes

- **Instagram Feed**: Focuses on elegance and key benefits (Maps, RSVP, Cost).
- **TikTok/Story**: Focuses on the "problem/solution" hook and high energy visuals.
